# hscredit 开发路线图

本文档描述了 hscredit 项目的开发计划和里程碑。

---

## 🎯 项目目标

将 scorecardpipeline 迁移到公司级开源项目 hscredit,去除第三方风控库依赖,提供更易用的 API 和完善的文档。
